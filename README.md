# Typescript - October 2025
SoftUni Typescript course

### 1️⃣ Install TypeScript
```bash
npm install -g typescript
```
### 2️⃣ Initialize TypeScript Configuration

```bash
tsc --init
```
This will generate a tsconfig.json file with the default TypeScript configuration.

### ▶️ Running TypeScript Files

### 1️⃣ Navigate to the Desired Folder

Move to the folder containing the .ts file you want to run.

### 2️⃣ Run the File with ts-node

```bash
ts-node fileName.ts
```
